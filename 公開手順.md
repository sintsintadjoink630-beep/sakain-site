# GitHub Pagesでの公開手順

## 1. GitHubアカウントを作る
https://github.com/ にアクセスし、無料アカウントを作成します。

## 2. リポジトリを作る
1. 右上の「+」→「New repository」
2. Repository name に `sakain-site` など任意の名前を入力（公開設定は Public）
3. 「Create repository」をクリック

## 3. このフォルダ内のファイルをアップロード
作成した以下のファイル・フォルダをそのままアップロードします。

```
index.html
mokuji.html
shop.html
makers.html
contact.html
css/style.css
js/main.js
products/drape.html
products/lace.html
products/roll-screen.html
products/blind.html
images/（写真を入れるフォルダ）
```

アップロード方法：リポジトリ画面の「Add file」→「Upload files」から、フォルダごとドラッグ＆ドロップしてください。

## 4. GitHub Pagesを有効にする
1. リポジトリの「Settings」タブを開く
2. 左メニューの「Pages」を選択
3. 「Branch」を `main`、フォルダを `/(root)` に設定して「Save」

数分後、`https://(あなたのアカウント名).github.io/sakain-site/` で公開されます。

## 5. 独自ドメイン（sakain.com）を使いたい場合
現在のドメイン `sakain.com` の管理画面（ドメイン取得サービス）で、DNS設定にGitHub Pages用のレコードを追加すると、
今までと同じURLのまま新しいサイトを表示できます。設定方法はドメイン管理サービスにより異なるため、
契約しているサービス名がわかれば手順をご案内します。
